# 📌 College Dashboard System

## 🏫 Project Overview

This is a **College Dashboard Web Application** developed using **Python (Flask)**, **HTML, CSS, and JavaScript**.
The system provides an interactive and responsive dashboard for managing and displaying college-related information.

---

## ✨ Features

* 📄 Responsive Dashboard Design
* 🧭 Navigation Menu
* 📢 Notice Section
* 🏢 Departments Information
* 📅 Academic Details
* 📬 Contact Form
* 📱 Mobile-Friendly Layout
* 🎨 Modern UI with Smooth Animations

---

## 🛠️ Technologies Used

* **Python**
* **Flask Framework**
* **HTML5**
* **CSS3**
* **JavaScript**
* **Bootstrap (if used)**

## 🚀 How to Run the Project

### 1️⃣ Install Requirements

Make sure Python is installed.

Then install Flask:

```
pip install flask
```

### 2️⃣ Run the Application

```
python app.py
```

### 3️⃣ Open in Browser

```
http://127.0.0.1:5000/
```
